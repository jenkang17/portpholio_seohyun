회원가입

  • 사전 체크 리스트
  • 사용자 정보를 입력할 수 있어야 한다.
  • 입력칸이 비워진 상태로 다음 단계로 넘어갈 수 없고 적절한 메시지를 띄워 사용자에게 알려준다.
  • 아이디 중복확인을 하여 중복되는 아이디가 있으면 메시지를 띄워 사용자에게알려준다.

로그인

  • 사전 체크 리스트
  • 사용자가 아이디와 패스워드를 입력할 수 있어야 한다.
  • 입력 칸이 비워진 상태로 다음 단계로 넘어갈 수 없고 적절한 메시지를 띄워 사용자에게 알려준다.
  • 아이디, 비밀번호가 일치하지 않을 시 메시지를 띄워 사용자에게 알려준다.
  • 조건에 맞지 않는 아이디를 입력하면 적절한 메시지를 띄워 사용자에게 알려준다.
  • 등록된 회원이 아니면 회원가입 페이지로 갈 수 있는 버튼이 있거나 회원가입 페이지 이동 여부를 물어 페이지 이동을 한다.
  • 로그인시 일반회원 로그인과 관리자 로그인은 구분 되어야 한다.

관리자 모드

  • 사전 체크 리스트
  • 접근제어가 필요한 페이지 및 모듈들에는 모두 통제수단이 적용되어야 한다.
  • 관리자 아이디로 접속시 관리자 페이지로 이동할 수 있어야 한다.
  • 관리자 아이디로 접속시 수정, 삭제, 업데이트 할 수 있는 버튼이 생긴다.
  • 관리자 아이디에게만 게시물 수정, 삭제, 업데이트 권한이 부여된다.
